# 查看 test_0109 分支差异指南

## 方法 1：使用 Git 命令查看单个文件差异（推荐）

在终端中运行以下命令查看重要文件的差异：

### 1. 查看导航文本文件（你刚翻译过的）
```bash
git diff test_0109 -- backend/app/utils/navigation_text.py
```

### 2. 查看聊天接口文件
```bash
git diff test_0109 -- backend/app/api/endpoints/chat.py
```

### 3. 查看配置文件
```bash
git diff test_0109 -- backend/app/core/config.py
```

### 4. 查看图服务文件
```bash
git diff test_0109 -- backend/app/services/graph_service.py
```

### 5. 查看 AI 服务文件
```bash
git diff test_0109 -- backend/app/services/ai_service.py
```

### 6. 查看导航客户端文件
```bash
git diff test_0109 -- backend/app/services/navigation_client.py
```

### 7. 查看脚本文件
```bash
git diff test_0109 -- backend/scripts/import_nodes_and_edges.py
git diff test_0109 -- backend/scripts/import_edges_only.py
git diff test_0109 -- backend/scripts/export_node_coordinates.py
```

### 8. 查看其他重要文件
```bash
git diff test_0109 -- .gitignore
git diff test_0109 -- README.md
```

## 方法 2：使用 VS Code 查看（最简单）

1. **打开 VS Code**
2. **安装 GitLens 扩展**（如果还没安装）
3. **查看差异**：
   - 在 VS Code 左侧点击源代码管理图标
   - 点击分支名称（main）
   - 选择 "比较分支"
   - 输入 `test_0109`
   - 会显示所有文件差异列表
   - 点击文件名查看具体差异

## 方法 3：查看所有重要文件的统计信息

```bash
# 查看所有文件的变化统计（排除 node_modules 和 chroma）
git diff --stat test_0109 | findstr /V "node_modules chroma"
```

## 方法 4：查看新增的 add 目录

```bash
# 查看 add 目录下有什么文件
git ls-tree -r test_0109 --name-only | findstr "add"

# 查看 add 目录的详细内容
git show test_0109:add --stat
```

## 方法 5：选择性合并文件

查看差异后，如果决定合并某个文件：

```bash
# 从 test_0109 分支检出单个文件（会覆盖当前文件）
git checkout test_0109 -- 文件路径

# 例如：
git checkout test_0109 -- backend/app/services/ai_service.py
```

## 重要提示

1. **navigation_text.py** - 你刚翻译成英文，test_0109 可能是中文版本，**不要合并这个文件**，保留你的英文版本

2. **chat.py** - 你刚翻译成英文，test_0109 可能是中文版本，**谨慎合并**

3. **先查看差异再决定** - 不要直接合并，先看差异内容

4. **备份重要文件** - 合并前可以备份：
   ```bash
   cp backend/app/api/endpoints/chat.py backend/app/api/endpoints/chat.py.backup
   ```
